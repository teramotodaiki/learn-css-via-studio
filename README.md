# Learn CSS via STUDIO

STUDIO で作成したウェブページを用いて HTML/CSS を学ぶための Chrome 拡張

# How to use

- GitHub からソースコード一式を zip でダウンロード
- 解答してフォルダに展開
- Google Chrome の chrome://extensions をひらく
- 「パッケージ化されていない拡張機能を読み込む」でフォルダを選択する
- STUDIO のプレビューページをひらいて左下のボタンを押す
